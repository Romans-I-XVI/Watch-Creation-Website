HTML (Logo/Navigation)

Changing the logo

Changing out your logo is simple. I've included a photoshop template, logo.psd which is 328 pixels wide by 108 pixels tall. Just save over the current image/logo.png file and you are done. You can resize your logo to any dimensions and the theme will adjust automatically.

Logo Maximum Width: Any width

Logo Maximum Height: Any height





Adjust between 4 Navigation Styles

You can choose between 4 navigation styles that are demoed in the live preview by changing the container ID for the navigation.

  • Default Navigation - navigation-narrow
  • Navigation Wide - navigation-wide
  • Navigation Dark - navigation-narrow-dark
  • Navigation Dark Wide - navigation-wide-dark
	<div id="navigation-narrow">
		<nav>

For example changing the default navigation style to a dark one can be done like so:

	<div id="navigation-narrow-dark">
		<nav>


Marking a Navigation Item as Selected

The navigation is setup to be easily selectable. For example, the About page will show the About link in the navigation with a darker font color. Just add a class of "current-menu-item" to the navigation list item and it will show as selected. For example:

<li>
	<a href="index.html">Home</a>
</li>
<li class="current-menu-item">
	<a href="about.html">About</a>
</li>
<li>
	<a href="styles.html">Styles</a>
</li>
Adding in a sidebar

Adding a sidebar is demoed in the sidebar-right.html and sidebar-left.html pages. It involves adding:

  • Add in two div classes "sidebar-container" and "content-sidebar"
  • Then add in your sidebar below the content in the class of <div class="sidebar">
<div class="sidebar-container">
	
<div class="page-content content-sidebar">

Then be close those two divs and add the sidebar code: 

<div class="sidebar">
Moving your sidebar left

Moving the sidebar left just involves changing the classes:

<div class="sidebar-container container-left">
	
<div class="page-content content-sidebar-left">

Then the sidebar gets one additional class:

<div class="sidebar sidebar-left">


Aligning Images Left, Right, and Center

Any images that are on the website by default will align with the text. The Styles page shows examples of how to float the images right, center, or left. Add a div with a class of "aligncenter", "alignright", and "alignleft" to float any of the images. For example:

<img class="alignright" src="images/demo-content/photo.jpg" width="140" height="103" />



Creating buttons

The default button is created by just adding one class to a text link. All of the background images and hover effects are added automatically! For Example:

<a href="http://www.google.com" target="_blank" class="button">Default Button</a>

Tabbed Content

The tabbed content is very simple to create. You just need to unordered lists. THe first list is the tab labels while the second list is the content. The only three classes you need to reuse are class="tabs" and class="tabs-content". The class="active" refers to the selected tab.

<ul class="tabs">
	<li><a class="active" href="#simple">Simple</a></li>
	<li><a href="#lightweight">Lightweight</a></li>
	<li><a href="#mobileFriendly">Mobile</a></li>
</ul>
<ul class="tabs-content">
	<li class="active" id="simple">The tabs are clean and simple unordered-list markup and basic CSS.</li>
	<li id="lightweight">The tabs are cross-browser, but don't need a ton of hacky CSS or markup.</li>
	<li id="mobileFriendly">The tabs work like a charm even on mobile devices. If there are too many they will stack (it's not completely awesome, but still retains functionality).</li>
</ul>

		</li>
		<!-- END SLIDE -->

	</ul>

	<script type="text/javascript">
	jQuery(document).ready(function($) {
		$('#slider').anythingSlider({
			delay               : 3500,      // How long between slideshow transitions in AutoPlay mode (in milliseconds)
			animationTime       : 400,       // How long the slideshow transition takes (in milliseconds)
			buildArrows         : false,      // If true, builds the forwards and backwards buttons
			buildNavigation     : true,      // If true, buildsa list of anchor links to link to each panel

Twitter/Flickr

Twitter Embedding

You can easily embed tweets by finding visiting this url https://twitter.com/settings/widgets when you are logged into twitter.

Just create a new widget by clicking "Create New" on the top right.

Then you have options for the twitter widget that you can use such as light or dark.

Lastly you can click "Create Widget" and it will give you the code to paste onto your site.

NOTE: Your widget will only appear on it's domain. So if you are testing it on your localhost it will not work.

Additional customizations can be found here: https://dev.twitter.com/docs/embedded-timelines. FOr example we added a transparent background on the sidebar widget with no header using 'data-chrome="noheader transparent"'
